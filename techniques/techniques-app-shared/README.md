## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Make script executable

```bash
$ chmod a+rx my-script.sh
$ ./my-script.sh
```

### prerequisites

generate localhost private key and public certificate with **mkcert** tool
